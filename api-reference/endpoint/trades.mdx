## GET /trades

Retrieve trading history and statistics for a specific user.

### Query Parameters

<ParamField query="userId" type="string" required>
  Extension user identifier
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Whether the request was successful
</ResponseField>

<ResponseField name="trades" type="Trade[]" required>
  Array of user's trading transactions
</ResponseField>

<ResponseField name="stats" type="UserStats" required>
  User's trading statistics
</ResponseField>

<ResponseField name="count" type="integer" required>
  Number of trades returned
</ResponseField>

### Trade Schema

<ResponseField name="id" type="string" required>
  Unique trade identifier
</ResponseField>

<ResponseField name="userId" type="string" required>
  Extension user identifier
</ResponseField>

<ResponseField name="type" type="string" required>
  Trade type: "buy" or "sell"
</ResponseField>

<ResponseField name="ticker" type="string" required>
  Memecoin ticker symbol
</ResponseField>

<ResponseField name="solAmount" type="number" required>
  SOL amount traded
</ResponseField>

<ResponseField name="timestamp" type="number" required>
  Trade timestamp (Unix milliseconds)
</ResponseField>

<ResponseField name="marketCap" type="number">
  Memecoin market cap at trade time
</ResponseField>

<ResponseField name="solPnL" type="number">
  Profit/loss for this trade in SOL
</ResponseField>

### UserStats Schema

<ResponseField name="totalTrades" type="integer" required>
  Total number of trades executed
</ResponseField>

<ResponseField name="winRate" type="number" required>
  Percentage of profitable trades (0-1)
</ResponseField>

<ResponseField name="totalPnL" type="number" required>
  Total profit/loss in SOL
</ResponseField>

<ResponseField name="totalVolume" type="number" required>
  Total SOL volume traded
</ResponseField>

### Example Request

```bash
curl -X GET "https://genny.lol/api/trades?userId=ext_123456"
```

### Example Response

```json
{
  "success": true,
  "trades": [
    {
      "id": "trade_123456",
      "userId": "ext_123456",
      "type": "buy",
      "ticker": "MOON",
      "solAmount": 10.5,
      "timestamp": 1699123456789,
      "marketCap": 50000,
      "solPnL": 1.5
    }
  ],
  "stats": {
    "totalTrades": 25,
    "winRate": 0.75,
    "totalPnL": 150.25,
    "totalVolume": 500.0
  },
  "count": 25
}
```

## POST /trades

Log a new simulated trade transaction.

### Request Body

<ParamField body="userId" type="string" required>
  Extension user identifier
</ParamField>

<ParamField body="sessionId" type="string" required>
  Session identifier for trade validation
</ParamField>

<ParamField body="timestamp" type="number">
  Trade timestamp (defaults to current time)
</ParamField>

<ParamField body="type" type="string" required>
  Trade type: "buy" or "sell"
</ParamField>

<ParamField body="ticker" type="string" required>
  Memecoin ticker symbol
</ParamField>

<ParamField body="address" type="string">
  Memecoin contract address
</ParamField>

<ParamField body="solAmount" type="number" required>
  SOL amount traded
</ParamField>

<ParamField body="marketCap" type="number">
  Memecoin market cap at trade time
</ParamField>

<ParamField body="solPrice" type="number">
  SOL price per token
</ParamField>

<ParamField body="solProceeds" type="number">
  SOL received from trade
</ParamField>

<ParamField body="solPnL" type="number">
  Profit/loss in SOL
</ParamField>

<ParamField body="balance" type="number">
  User's SOL balance after trade
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Whether the trade was logged successfully
</ResponseField>

<ResponseField name="trade" type="Trade" required>
  The logged trade record
</ResponseField>

<ResponseField name="message" type="string" required>
  Success confirmation message
</ResponseField>

### Example Request

```bash
curl -X POST "https://genny.lol/api/trades" \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "ext_123456",
    "sessionId": "session_789",
    "type": "buy",
    "ticker": "MOON",
    "address": "7xKXtg2CW99PHZdq3gTtD6zWjTJEe1Z4y8K8Z4v7v7",
    "solAmount": 10.5,
    "marketCap": 50000,
    "solPrice": 0.000001,
    "solProceeds": 12.0,
    "solPnL": 1.5,
    "balance": 989.5
  }'
```

### Example Response

```json
{
  "success": true,
  "trade": {
    "id": "trade_123456",
    "userId": "ext_123456",
    "type": "buy",
    "ticker": "MOON",
    "solAmount": 10.5,
    "timestamp": 1699123456789,
    "marketCap": 50000,
    "solPnL": 1.5
  },
  "message": "Trade logged successfully"
}
```
