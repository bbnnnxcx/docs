## GET /leaderboard

Retrieve the current trading leaderboard with user rankings and global statistics.

### Query Parameters

<ParamField query="timeFilter" type="string" default="all-time">
  Time period for leaderboard rankings. Options: `daily`, `weekly`, `monthly`, `all-time`
</ParamField>

<ParamField query="limit" type="integer" default="100">
  Maximum number of leaderboard entries to return (1-1000)
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Whether the request was successful
</ResponseField>

<ResponseField name="leaderboard" type="LeaderboardEntry[]" required>
  Array of leaderboard entries
</ResponseField>

<ResponseField name="stats" type="GlobalStats" required>
  Global platform statistics
</ResponseField>

<ResponseField name="timeFilter" type="string" required>
  The time filter used for this request
</ResponseField>

<ResponseField name="count" type="integer" required>
  Number of entries returned
</ResponseField>

<ResponseField name="payout" type="PayoutInfo" required>
  Information about payout timing
</ResponseField>

### LeaderboardEntry Schema

<ResponseField name="rank" type="integer" required>
  User's current ranking position
</ResponseField>

<ResponseField name="username" type="string" required>
  User's display name
</ResponseField>

<ResponseField name="pnl" type="number" required>
  Total profit/loss in SOL
</ResponseField>

<ResponseField name="winRate" type="number" required>
  Percentage of profitable trades (0-1)
</ResponseField>

<ResponseField name="totalVolume" type="number" required>
  Total SOL traded
</ResponseField>

<ResponseField name="xUsername" type="string">
  Twitter/X username (if connected)
</ResponseField>

### Example Request

```bash
curl -X GET "https://genny.lol/api/leaderboard?timeFilter=weekly&limit=50"
```

### Example Response

```json
{
  "success": true,
  "leaderboard": [
    {
      "rank": 1,
      "username": "trader123",
      "pnl": 250.50,
      "winRate": 0.75,
      "totalVolume": 1500.00,
      "xUsername": "trader123"
    }
  ],
  "stats": {
    "totalUsers": 1250,
    "totalTrades": 45670,
    "totalVolume": 125000.00
  },
  "timeFilter": "weekly",
  "count": 50,
  "payout": {
    "nextPayoutTime": 1699123456789,
    "isPayoutDue": false,
    "timeUntilPayout": 14400000
  }
}
```
