## POST /profile/create

Create a new user profile with username and wallet address for paper trading.

### Request Body

<ParamField body="username" type="string" required>
  Desired username (3-20 characters, alphanumeric + underscore only)
</ParamField>

<ParamField body="walletAddress" type="string" required>
  Solana wallet address for profile verification
</ParamField>

<ParamField body="xUsername" type="string">
  Optional Twitter/X username for social features
</ParamField>

<ParamField body="extensionUserId" type="string">
  Chrome extension user identifier for linking
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Whether the profile was created successfully
</ResponseField>

<ResponseField name="otpCode" type="string" required>
  One-time password for extension linking
</ResponseField>

<ResponseField name="profile" type="Profile" required>
  Created profile information
</ResponseField>

### Profile Schema

<ResponseField name="id" type="string" required>
  Unique profile identifier
</ResponseField>

<ResponseField name="username" type="string" required>
  User's chosen username
</ResponseField>

<ResponseField name="walletAddress" type="string" required>
  Associated Solana wallet address
</ResponseField>

### Example Request

```bash
curl -X POST "https://genny.lol/api/profile/create" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "trader123",
    "walletAddress": "7xKXtg2CW99PHZdq3gTtD6zWjTJEe1Z4y8K8Z4v7v7",
    "xUsername": "trader123",
    "extensionUserId": "ext_123456"
  }'
```

### Example Response

```json
{
  "success": true,
  "otpCode": "123456",
  "profile": {
    "id": "user_123456",
    "username": "trader123",
    "walletAddress": "7xKXtg2CW99PHZdq3gTtD6zWjTJEe1Z4y8K8Z4v7v7"
  }
}
```

### Error Responses

<AccordionGroup>
  <Accordion title="400 Bad Request">
    ```json
    {
      "error": "Username must be 3-20 characters (letters, numbers, underscore only)"
    }
    ```
  </Accordion>

  <Accordion title="409 Conflict">
    ```json
    {
      "error": "Username already taken"
    }
    ```
  </Accordion>
</AccordionGroup>
