## GET /health

Check if the Genny API is running and responsive.

### Response

<ResponseField name="status" type="string" required>
  API status indicator
</ResponseField>

<ResponseField name="timestamp" type="number" required>
  Current server timestamp (Unix milliseconds)
</ResponseField>

### Example Request

```bash
curl -X GET "https://genny.lol/api/health"
```

### Example Response

```json
{
  "status": "ok",
  "timestamp": 1699123456789
}
```

### Error Responses

<AccordionGroup>
  <Accordion title="500 Internal Server Error">
    ```json
    {
      "error": "Internal server error"
    }
    ```
  </Accordion>
</AccordionGroup>
